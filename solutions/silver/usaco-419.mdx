## Solution 1 (Binary Search, Prefix Sum, and Math)

### Explanation

We can binary search on real values to find the minimum average $X$ we can achieve.

Now, if we find the prefix sum array $\texttt{pref}$ of the machine milk production array $M$, consider the condition we need to achieve an average $\le X$ by removing the subarray $(i, j]$:
$$
\frac{S-\left(\texttt{pref}[j]-\texttt{pref}[i]\right)}{N-\left(j-i\right)}\le X,
$$
where $S$ is the current sum of the array. After multiplying both sides by the denominator and isolating all terms involving $i$, we get:
$$
\texttt{pref}[i]-X\cdot i\le X\cdot\left(N-j\right)-S+\texttt{pref}[j].
$$
For a fixed $j$, we should try to minimize the left side expression. So we can iterate $j$ from $2$ to $N - 1$ and keep a running minimum of the left side of the inequality. If the condition is satisfied, we can achieve an average $\le X$.

### Implementation

**Time Complexity:** $\mathcal{O}(N\log\left(\frac{1}{\varepsilon}\right))$, where $\varepsilon$ is the fixed error margin.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	freopen("sabotage.in", "r", stdin);
	freopen("sabotage.out", "w", stdout);

	int n;
	cin >> n;
	vector<int> m(n);
	for (int i = 0; i < n; i++) cin >> m[i];

	// Prefix sums on the milk production array m
	vector<long long> pref_sum(n + 1);
	for (int i = 0; i < n; i++) { pref_sum[i + 1] = pref_sum[i] + m[i]; }

	long long arr_sum = pref_sum[n];

	// Check function to see if average x can be achieved
	auto check = [&](double x) {
		double left_side = pref_sum[1] - x;
		for (int j = 2; j <= n - 1; j++) {
			double right_side = x * (n - j) - arr_sum + pref_sum[j];
			if (left_side <= right_side) { return true; }
			left_side = min(left_side, pref_sum[j] - x * j);
		}
		return false;
	};

	double l = 1, r = 10000;

	while (r - l > 1e-4) {
		double mid = (l + r) / 2;
		if (check(mid)) r = mid;
		else l = mid;
	}

	cout << fixed << setprecision(3) << l << "\n";
}
```

## Solution 2 (Binary Search, Maximum Sum Subarray with DP)

As before, we can binary search on real values to find the minimum average $X$ we can achieve.

Note that we now must find if a specific average is possible in $\mathcal O(n)$ time.
However, the average is dependent on the number of stations that are not unplugged, which makes this difficult in $\mathcal O(n)$.
But since we already know the target average, we can temporarily subtract each number in the list by the target average;
now, the sign of the sum of a set of stations will tell us whether their average is equal to or below the target ($<=0$) or above the target ($>0$).

To have the best chance at achieving the average, the sum of the remaining elements in the temporary array after the sabotage should be at a minimum;
this means the sum of the sabotaged elements, a continuous range in $[2, n-1]$ should be at a maximum.

The maximum continuous range in an list could be found using a standard DP algorithm:
Let $K_i$ be the maximum sum of a range ending at index $2\leq i \leq N-1$.
The transitions are $K_i=\max(K_{i-1}+M_i,M_i)$, as either a new range can be began at $i$, or a previous range may be continued.
In this problem, we initialize $K_2=M_2$, as there can only be a 1-element array ending with index 2, and transition until $K_{N-1}$.
The maximum subarray sum will be the maximum of all elements from $K_2$ to $K_{N-1}$.

We check if the sum of the entire array minus the sum of the maximum subarray range is nonpositive, and if so, the check passes.

```cpp
#include <fstream>
#include <iomanip>
using namespace std;
constexpr int MAX_N = 1e5;
double M[MAX_N + 1];
double diff[MAX_N + 1];
double K[MAX_N + 1];
int n;

// Check whether an average is possible
bool check(double x) {
	double total = 0;
	// Compute the temporary difference array
	// and compute its total sum
	for (int i = 1; i <= n; i++) {
		diff[i] = M[i] - x;
		total += diff[i];
	}

	// Compute each K[i] and find the maximum of the elements
	K[2] = diff[2];
	double max_sum = K[2];
	for (int i = 3; i < n; i++) {
		K[i] = max(K[i - 1] + diff[i], diff[i]);
		max_sum = max(max_sum, K[i]);
	}
	return total - max_sum < 1e-6;
}
int main() {
	ifstream fin("sabotage.in");
	fin >> n;
	for (int i = 1; i <= n; i++) fin >> M[i];
	double l = 0, r = 10000;
	while (l + 1e-4 < r) {
		double mid = (l + r) / 2;
		if (check(mid)) r = mid;
		else l = mid;
	}
	ofstream fout("sabotage.out");
	fout << fixed << setprecision(3) << l << endl;
	return 0;
}
```
