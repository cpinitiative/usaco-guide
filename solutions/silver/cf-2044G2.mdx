## Explanation

First, note the structure of the graph. Every node has a singular outgoing edge, making it a functional graph. As a result, the graph consists of several connected components, each containing exactly one cycle, with all the nodes either in the cycle or feeding into it.

Notice that if a node is not in the cycle, the process will not become stable until that node's plushie reaches the cycle. Consider the nodes along a path leading into a cycle. On each day, the spiders will pass one of their plushies (if they have one) to $r_i$. However, some of the spiders that send a plushie will not receive one in return, causing the system to have changed numbers. Thus, like the easy version, once every plushie reaches a cycle, each spider will both receive and give a plushie, keeping the system stable.

Unlike the easy version, spiders can now hold multiple plushies. Model each connected component as a cycle with rooted trees attached to its nodes, where each tree is rooted at a node on the cycle. So, how long will it take for some spider $i$ in a tree to give away all of its plushies? This is equal to the subtree size of $i$, since spider $i$ will give away exactly one plushie every year, and all the plushies in the subtree of $i$ must pass through $i$.

Additionally, we can think of the time it takes through two bounding factors. The time it would take for all the plushies to flow into a cycle through entrance $i$, and the max time it would take for a plushie to reach the cycle (max depth). Since the subtree size is always greater than or equal to the max depth, we can simply take the max subtree size as the answer.

By constructing a separate adjacency list, we can run a DFS on each of these trees, allowing us to compute all the subtree sizes, ignoring the root (which is part of the cycle). The final answer will be the maximum subtree size among all such nodes, as that is the maximum time it takes for all the plushies to flow into cycles.


## Implementation

**Time Complexity:** $\mathcal{O}(N)$

```cpp

#include <bits/stdc++.h>
using namespace std;

void solve() {
	int n;
	cin >> n;
	vector<int> a(n);
	vector<vector<int>> adj(n);
	for (int i = 0; i < n; i++) {
		cin >> a[i];
		a[i]--;
		adj[a[i]].push_back(i);
	}

	vector<int> counts(n);
	vector<bool> visited(n);
	int ans = 2;

	// DFS that traverses through the trees and returns subtree sizes
	auto dfs = [&](auto &&self, int node) -> int {
		int mx = 0;
		visited[node] = true;
		for (int next_node : adj[node]) {
			if (counts[next_node] != 2) { mx += self(self, next_node); }
		}
		return mx + 1;
	};

	for (int i = 0; i < n; i++) {
		if (!visited[i]) {
			int node = i;
			vector<int> nodes;

			// Get all the nodes in the cycle
			while (counts[node] != 2) {
				visited[node] = true;
				counts[node]++;
				if (counts[node] == 2) nodes.push_back(node);
				node = a[node];
			}

			// Run the DFS on each tree rooted in the cycle
			int cur = 0;
			for (int j = 0; j < nodes.size(); j++) {
				for (int root : adj[nodes[j]]) {
					if (counts[root] != 2) {
						int subtree_size = dfs(dfs, root);
						cur = max(subtree_size + 2, cur);
					}
				}
			}
			ans = max(ans, cur);
		}
	}
	cout << ans << '\n';
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(NULL);
	int t;
	cin >> t;
	while (t--) { solve(); }
}
```
