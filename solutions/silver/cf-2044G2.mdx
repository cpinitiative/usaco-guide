## Explanation

First, note the structure of the graph. Every node has a singular outgoing edge, making it a functional graph. As a result, the graph consists of several connected components, each containing exactly one cycle, with all the nodes either in the cycle or feeding into it.

Notice that if a node is not in the cycle, the process will not become stable until that plushie reaches into the cycle. Consider the nodes along a path leading into a cycle. On each day, the spiders will pass one of their plushies (if they have one) to $r_i$. However, some of the spiders that send a plushie will not recieve one in return, causing the system to have changed numbers. Thus, like the easy version, once every plushie reaches a cycle, each spider will both recieve and give a plushie, keeping the system stable.

Unlike the easy version, spiders can now hold multiple plushies. Model each connected component as a cycle with rooted trees attached to its nodes, where each tree is rooted at a node on the cycle. So, how long will it take for some spider $i$ in a tree to give away all of it's plushies? This is equal to the subtree size of $i$, since spider $i$ will give away exactly one plushie every year, and all the plushies in the subtree of $i$ will must pass through $i$.

By constructing a seperate adjacency list, we can run a DFS on each of these trees, allowing us to compute all the subtree sizes, ignoring the root (which is part of the cycle). The final answer will be the maximum subtree size among all such nodes.


## Implementation

**Time Complexity:** $\mathcal{O}(N)$

```cpp

#include <bits/stdc++.h>
using namespace std;
void solve() {
	int n;
	cin >> n;
	vector<int> a(n);
	vector<vector<int>> c(n);
	for (int i = 0; i < n; i++) {
		cin >> a[i];
		a[i]--;
		c[a[i]].push_back(i);
	}

	vector<int> b(n);
	vector<bool> v(n);
	int ans = 2;

	// DFS that traverses through the trees and returns subtree sizes
	auto dfs = [&](auto &&self, int nd) -> int {
		int mx = 0;
		v[nd] = true;
		for (int i = 0; i < c[nd].size(); i++) {
			if (b[c[nd][i]] != 2) { mx += self(self, c[nd][i]); }
		}
		return mx + 1;
	};

	for (int i = 0; i < n; i++) {
		if (!v[i]) {
			int nd = i;
			vector<int> tm;

			// Get all the nodes in the cycle
			while (b[nd] != 2) {
				v[nd] = true;
				b[nd]++;
				if (b[nd] == 2) tm.push_back(nd);
				nd = a[nd];
			}

			// Run the DFS on each tree rooted in the cycle
			int cur = 0;
			for (int j = 0; j < tm.size(); j++) {
				for (int z = 0; z < c[tm[j]].size(); z++) {
					if (b[c[tm[j]][z]] != 2) {
						int d = dfs(dfs, c[tm[j]][z]);
						cur = max(d + 2, cur);
					}
				}
			}
			ans = max(ans, cur);
		}
	}
	cout << ans << '\n';
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(NULL);
	int t;
	cin >> t;
	while (t--) { solve(); }
}
```
