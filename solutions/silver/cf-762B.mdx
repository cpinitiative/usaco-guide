## Explanation

The main idea is to notice that **universal ports are more flexible** than USB-only or PS/2-only ports.

Because they can take either type, we don’t want to use them too early. If we assign a cheap mouse to a universal port right away, we might later be forced to use a more expensive mouse for a restricted port, which is avoidable.

So it makes sense to deal with the **restricted ports first**.

Split the mice into two arrays:
- `usbCosts`
- `ps2Costs`

Sort both arrays. Then:
- take the cheapest `usbPorts` USB mice  
- take the cheapest `ps2Ports` PS/2 mice  

At this stage, we’ve used the restricted ports as efficiently as possible.

Now look at the remaining mice: universal ports don’t care about type anymore, so we just want the cheapest options overall.

- put all unused mice into one array  
- sort it  
- take the cheapest `universalPorts` mice  

### Why this works

The key is delaying the use of flexible ports.

If an optimal solution ever uses a universal port for a mouse while a restricted port is filled with a more expensive mouse of the same type, we can swap them. The cheaper mouse goes to the restricted port, and the more expensive one can be moved or discarded. This never reduces the number of mice used and never increases the cost.

So there is always an optimal solution where we fill restricted ports first with the cheapest possible mice, and only then use universal ports.

### Alternative approach

The editorial also describes a two pointers approach. Instead of directly filling ports greedily, it considers how many USB and PS/2 mice to take and adjusts the counts while maintaining feasibility with the available ports.

This leads to a solution where we iterate over the number of USB mice taken and adjust the number of PS/2 mice accordingly, ensuring that the total number of mice fits within the available USB, PS/2, and universal ports.

We omit the full implementation here since the greedy approach is simpler and sufficient.

## Implementation

**Time Complexity:** $\mathcal{O}(M \log M)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int usbPorts, ps2Ports, universalPorts;
    cin >> usbPorts >> ps2Ports >> universalPorts;

    int m;
    cin >> m;

    vector<long long> usbCosts, ps2Costs;

    for (int i = 0; i < m; i++) {
        long long cost;
        string type;
        cin >> cost >> type;

        if (type == "USB") {
            usbCosts.push_back(cost);
        } else {
            ps2Costs.push_back(cost);
        }
    }

    sort(usbCosts.begin(), usbCosts.end());
    sort(ps2Costs.begin(), ps2Costs.end());

    long long totalCost = 0;
    int totalMice = 0;

    vector<long long> remaining;

    // Fill USB-only ports
    for (int i = 0; i < min((int)usbCosts.size(), usbPorts); i++) {
        totalCost += usbCosts[i];
        totalMice++;
    }

    for (int i = usbPorts; i < (int)usbCosts.size(); i++) {
        remaining.push_back(usbCosts[i]);
    }

    // Fill PS/2-only ports
    for (int i = 0; i < min((int)ps2Costs.size(), ps2Ports); i++) {
        totalCost += ps2Costs[i];
        totalMice++;
    }

    for (int i = ps2Ports; i < (int)ps2Costs.size(); i++) {
        remaining.push_back(ps2Costs[i]);
    }

    // Use universal ports
    sort(remaining.begin(), remaining.end());

    for (int i = 0; i < min((int)remaining.size(), universalPorts); i++) {
        totalCost += remaining[i];
        totalMice++;
    }

    cout << totalMice << " " << totalCost << "\n";
}
```
