## Explanation

Universal ports are more flexible than restricted ones—they can accept either type. So we should avoid using them unless necessary.

If we assign a mouse that could go to a restricted port into a universal port, we might later be forced to use a more expensive mouse for that restricted port.


### Greedy Strategy

- Sort all mice by cost
- For each mouse:
  - If a matching restricted port is available, use it
  - Otherwise, use a universal port (if available)
- After all restricted ports are filled, assign the cheapest remaining mice to universal ports


### Why this works

The key is delaying the use of flexible ports.

If a solution assigns a cheap mouse to a universal port while a restricted port uses a more expensive mouse of the same type, we can swap them. This does not increase cost and keeps the number of assigned mice the same.

So there exists an optimal solution where restricted ports are filled first with the cheapest valid mice, and universal ports are used only afterward.

## Implementation

**Time Complexity:** $\mathcal{O}(M \log M)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	int usbPorts, ps2Ports, universalPorts;
	cin >> usbPorts >> ps2Ports >> universalPorts;

	int m;
	cin >> m;

	vector<long long> usbCosts, ps2Costs;

	for (int i = 0; i < m; i++) {
		long long cost;
		string type;
		cin >> cost >> type;

		if (type == "USB") {
			usbCosts.push_back(cost);
		} else {
			ps2Costs.push_back(cost);
		}
	}

	sort(usbCosts.begin(), usbCosts.end());
	sort(ps2Costs.begin(), ps2Costs.end());

	long long totalCost = 0;
	int totalMice = 0;

	vector<long long> remaining;

	// Fill USB-only ports
	for (int i = 0; i < min((int)usbCosts.size(), usbPorts); i++) {
		totalCost += usbCosts[i];
		totalMice++;
	}

	for (int i = usbPorts; i < (int)usbCosts.size(); i++) {
		remaining.push_back(usbCosts[i]);
	}

	// Fill PS/2-only ports
	for (int i = 0; i < min((int)ps2Costs.size(), ps2Ports); i++) {
		totalCost += ps2Costs[i];
		totalMice++;
	}

	for (int i = ps2Ports; i < (int)ps2Costs.size(); i++) {
		remaining.push_back(ps2Costs[i]);
	}

	// Use universal ports
	sort(remaining.begin(), remaining.end());

	for (int i = 0; i < min((int)remaining.size(), universalPorts); i++) {
		totalCost += remaining[i];
		totalMice++;
	}

	cout << totalMice << " " << totalCost << "\n";
}
```

# Solution 2 (Two Pointers)

## Explanation

The editorial also describes a two pointers approach. Instead of directly filling ports greedily, it considers how many USB and PS/2 mice to take and adjusts the counts while maintaining feasibility with the available ports.

This leads to a solution where we iterate over the number of USB mice taken and adjust the number of PS/2 mice accordingly, ensuring that the total number of mice fits within the available USB, PS/2, and universal ports.

We omit the full explanation here since the greedy approach is simpler and sufficient.

---

## Implementation

**Time Complexity:** $\mathcal{O}(M \log M)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int A, B, C; // A = USB ports, B = PS/2 ports, C = universal ports
    cin >> A >> B >> C;

    int M;
    cin >> M;

    vector<long long> usb, ps2;

    // Separate mice by type
    for (int i = 0; i < M; i++) {
        long long cost;
        string type;
        cin >> cost >> type;
        if (type == "USB") usb.push_back(cost);
        else ps2.push_back(cost);
    }

    // Sort costs so we always take cheapest first
    sort(usb.begin(), usb.end());
    sort(ps2.begin(), ps2.end());

    int R = ps2.size(); // current number of PS/2 mice taken
    long long sum = 0;

    // Initially take all PS/2 mice
    for (auto x : ps2) sum += x;

    // (number of mice, negative cost) for lexicographic max
    pair<int, long long> ans = {0, 0};

    // Try taking L USB mice
    for (int L = 0; L <= (int)usb.size(); L++) {
        // Add the next USB mouse (if L > 0)
        if (L > 0) sum += usb[L - 1];

        // Check if current (L, R) fits in available ports
        auto can_form = [&]() {
            return L <= A + C &&       // USB + universal can handle USB mice
                   R <= B + C &&       // PS/2 + universal can handle PS/2 mice
                   L + R <= A + B + C; // total ports constraint
        };

        bool bad = false;

        // If not feasible, reduce R (drop most expensive PS/2 mice)
        while (!can_form()) {
            if (R == 0) {
                bad = true;
                break;
            }
            R--;
            sum -= ps2[R]; // remove largest remaining PS/2 cost
        }

        if (bad) break;

        // Maximize number of mice, then minimize cost
        ans = max(ans, {L + R, -sum});
    }

    cout << ans.first << " " << -ans.second << "\n";
}
```
