## Explanation

We are given:
- `usbPorts`: USB-only ports  
- `ps2Ports`: PS/2-only ports  
- `universalPorts`: ports that accept either type  

Each mouse:
- has a type (USB or PS/2)
- has a cost  

Our goal is to **buy the maximum number of mice while minimizing total cost**.

---

### Fill USB and PS/2 ports first

A natural question is: *should we use universal ports immediately since they are more flexible?*

At first glance, that might seem reasonable. However, doing so can lead to suboptimal choices. A universal port could take either type, but a USB port can only take a USB mouse. If we use a universal port too early, we might “waste” a cheap mouse that could have filled a restricted port instead.

So the safer strategy is:
- First handle the **restricted ports**, where choices are limited.

We separate all mice into:
- `usbCosts`
- `ps2Costs`

After sorting both arrays:
- Take the cheapest `usbPorts` USB mice  
- Take the cheapest `ps2Ports` PS/2 mice  

There is no reason to skip a cheaper mouse here, since doing so would only increase total cost.

---

### Use universal ports for the cheapest remaining mice

After filling restricted ports, some mice remain unused.

Now the situation changes:
- Universal ports have no restriction on type  
- We are free to choose any remaining mouse  

At this point, the only thing that matters is **cost**.

So we:
- Combine all remaining USB and PS/2 mice  
- Sort them  
- Take the cheapest `universalPorts` mice  

---

### Why this greedy works

The key is that we always make decisions under the tightest constraints first:
- Restricted ports are handled first because they limit our choices  
- Flexible ports are handled later, when we have full freedom  

If we reversed this order, we could end up assigning cheap mice to flexible ports and being forced to use more expensive ones later.

This approach maximizes the number of mice used while keeping the total cost as small as possible.
---

## Implementation

**Time Complexity:** $\mathcal{O}(M \log M)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int usbPorts, ps2Ports, universalPorts;
    cin >> usbPorts >> ps2Ports >> universalPorts;

    int m;
    cin >> m;

    vector<long long> usbCosts, ps2Costs;

    for (int i = 0; i < m; i++) {
        long long cost;
        string type;
        cin >> cost >> type;

        if (type == "USB") {
            usbCosts.push_back(cost);
        } else {
            ps2Costs.push_back(cost);
        }
    }

    sort(usbCosts.begin(), usbCosts.end());
    sort(ps2Costs.begin(), ps2Costs.end());

    long long totalCost = 0;
    int totalMice = 0;

    vector<long long> remaining;

    // Fill USB-only ports
    for (int i = 0; i < min((int)usbCosts.size(), usbPorts); i++) {
        totalCost += usbCosts[i];
        totalMice++;
    }

    for (int i = usbPorts; i < (int)usbCosts.size(); i++) {
        remaining.push_back(usbCosts[i]);
    }

    // Fill PS/2-only ports
    for (int i = 0; i < min((int)ps2Costs.size(), ps2Ports); i++) {
        totalCost += ps2Costs[i];
        totalMice++;
    }

    for (int i = ps2Ports; i < (int)ps2Costs.size(); i++) {
        remaining.push_back(ps2Costs[i]);
    }

    // Use universal ports
    sort(remaining.begin(), remaining.end());

    for (int i = 0; i < min((int)remaining.size(), universalPorts); i++) {
        totalCost += remaining[i];
        totalMice++;
    }

    cout << totalMice << " " << totalCost << "\n";
}
```
