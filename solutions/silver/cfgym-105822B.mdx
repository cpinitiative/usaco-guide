## Explanation

We must obtain $2$ pairs of equal numbers.
To make two indices $i_1$ and $i_2$ contain the same value, $|a[i_1]-a[i_2]|$ operations are needed.
Therefore, it is optimal to sort the array and make adjacent pair of numbers equal.

Thus, we sort the input array, and compute $d[i]=a[i+1]-a[i]$ for $0<i<n-1$.
The problem now reduces to choosing two $i$'s $i_1$ and $i_2$ such that $d[i_1]+d[i_2]$ is minimized.
However, **the two pairs cannot share any number**, so $i_1$ and $i_2$ cannot be adjacent
to each other (if $i_2=i_1+1$, the pairs would be $i_1, i_1+1$ and $i_1+1, i_1+2$, which overlap).

Without loss of generality, $i_1 < i_2$.
We loop over $2\leq i_2<n-1$ (since $i_2=1$ is impossible), so $d[i_2]$ is fixed;
$i_1$ must be in $1\leq i_1<i_2-1$, so we maintain a prefix minimum of $d[i_1]$ cumulatively
to obtain the best $d[i_1]$ for each $i_2$.

## Implementation

**Time Complexity:** $\mathcal{O}(N \log N)$

```cpp
#include <algorithm>
#include <climits>
#include <iostream>
#include <vector>

using namespace std;

void solve() {
	int n;
	cin >> n;

	vector<int> sticks(n);
	for (int i = 0; i < n; i++) cin >> sticks[i];
	sort(sticks.begin(), sticks.end());

	vector<int> d(n);
	for (int i = 0; i < n - 1; i++) d[i] = sticks[i + 1] - sticks[i];

	int ans = INT_MAX;
	int min_previous_diff = d[0];
	// Loop over i_2
	for (int i = 2; i < n - 1; i++) {
		ans = min(ans, d[i] + min_previous_diff);
		// d[i+1] can be combined with anything before d[i].
		min_previous_diff = min(min_previous_diff, d[i - 1]);
	}
	cout << ans << endl;
}

int main() {
	int test_num;
	cin >> test_num;
	for (int i = 0; i < test_num; i++) solve();
	return 0;
}
```
