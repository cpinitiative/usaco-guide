## Explanation

We must obtain $2$ pairs of equal numbers.
To make two indices $i_1$ and $i_2$ contain the same value, $|sticks[i_1]-sticks[i_2]|$ operations are needed.
Therefore, it is optimal to sort the array and make adjacent pairs of numbers equal.

Thus, we sort the input array, and compute $diff[i]=sticks[i+1]-sticks[i]$ for $0<i<n-1$.
The problem now reduces to choosing two indices $i_1$ and $i_2$ such that $diff[i_1]+diff[i_2]$ is minimized.
However, **the two pairs cannot share any number**, so $i_1$ and $i_2$ cannot be adjacent
to each other (if $i_2=i_1+1$, the pairs would be $i_1, i_1+1$ and $i_1+1, i_1+2$, which overlap).

Without loss of generality, $i_1 < i_2$.
We loop over $2\leq i_2<n-1$ (since $i_2=1$ is impossible). By fixing $diff[i_2], i_1$ must be in $1\leq i_1<i_2-1$, and we can maintain a prefix minimum of $diff[i_1]$ to obtain the best $diff[i_1]$ for each $i_2$.

## Implementation

**Time Complexity:** $\mathcal{O}(N \log N)$

```cpp
#include <algorithm>
#include <climits>
#include <iostream>
#include <vector>

using namespace std;

void solve() {
	int n;
	cin >> n;

	vector<int> sticks(n);
	for (int i = 0; i < n; i++) cin >> sticks[i];
	sort(sticks.begin(), sticks.end());

	vector<int> diff(n - 1);
	for (int i = 0; i < n - 1; i++) diff[i] = sticks[i + 1] - sticks[i];

	int ans = INT_MAX;
	int min_previous_diff = diff[0];
	// Loop over i_2
	for (int i = 2; i < n - 1; i++) {
		ans = min(ans, diff[i] + min_previous_diff);
		// diff[i+1] can be combined with anything before diff[i].
		min_previous_diff = min(min_previous_diff, diff[i - 1]);
	}
	cout << ans << endl;
}

int main() {
	int test_num;
	cin >> test_num;
	for (int i = 0; i < test_num; i++) solve();
	return 0;
}
```
