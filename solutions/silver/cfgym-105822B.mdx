## Explanation

Sort the input array.
We must obtain $2$ pairs of the same number.
To form a pair, it is always suboptimal to choose any two numbers that are not adjacent to each other after sorting.
Thus, we compute $d[i]=a[i+1]-a[i]$, for $0<i<n-1$, and the problem reduces to choosing two $i$'s $i_1$ and $i_2$ such that $d[i_1]+d[i_2]$ is minimized.
However, **the two pairs cannot share any number**, so $i_1$ and $i_2$ cannot be adjacent to each other.

Without loss of generality, $i_1 < i_2$.
We loop over $2\leq i_2<n-1$ (since $i_2=1$ is impossible), so now $d[i_2]$ is fixed;
$i_1$ must be in $1\leq i_1<i_2-1$, so we maintain a prefix minimum of $d[i_1]$ cumulatively to obtain the best $d[i_1]$ for each $i_2$.

## Implementation

**Time Complexity:** $\mathcal{O}(N \log N)$

```cpp
#include <algorithm>
#include <climits>
#include <iostream>
#include <vector>
using namespace std;

void solve() {
	int n;
	cin >> n;

	vector<int> sticks(n);
	for (int i = 0; i < n; i++) cin >> sticks[i];
	sort(sticks.begin(), sticks.end());

	vector<int> d(n);
	for (int i = 0; i < n - 1; i++) d[i] = sticks[i + 1] - sticks[i];

	int ans = INT_MAX;
	int min_previous_diff = d[0];
	// Loop over i_2
	for (int i = 2; i < n - 1; i++) {
		ans = min(ans, d[i] + min_previous_diff);
		// d[i+1] can be combined with anything before d[i].
		min_previous_diff = min(min_previous_diff, d[i - 1]);
	}
	cout << ans << endl;
}

int main() {
	int test_cases;
	cin >> test_cases;
	for (int i = 0; i < test_cases; i++) solve();
	return 0;
}
```
