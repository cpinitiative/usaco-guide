## Explanation

We construct the directed graph with edges $a_i\to b_i$, and call each $b_i$ the neighbor of $a_i$.
Having each $a_i$ goes after its $b_i$ in the list results in a working configuration, as $b_i$ 
cannot directly follow $a_i$ when it is before $a_i$ in the first place.
This can always happen, except when the graph has cycles.
If there is a cycle in the graph, however, we can neglect an edge in a cycle. This is still valid, 
as the cycles have at least $3$ vertices.

To see why, consider this example: we require vertex A to be placed before B, B to be before C, and C to 
be before A; from the first two requirements, A must be before C with at least $1$ element (B) in between. 
Since A and C are not adjacent, their order does not matter, so we may neglect the third edge.

We use a post-order DFS to traverse all the nodes; at each node, we DFS all its neighbors before outputting the current node number.
This ensures that all neighbors of a vertex come before itself.
In case of encountering a cycle while traversing, we would revisit an already-visited vertex, which we can safely skip, as neglecting an edge is permissible.

Note that by DFS'ing each unvisited vertex in order, we may start visiting a neighbor of some other vertex 
we have not yet traversed; however, that other vertex would be traversed in a later DFS, so the order 
is still correct.  The later DFS will also skip the current neighbor, as it has already been traversed.

## Implementation

**Time complexity:** $\mathcal O(n+m)$

```cpp
// Based on solution by JuanMata on CodeForces
#include <iostream>
#include <vector>

using namespace std;

const int MAX_N = 30001;

vector<int> graph[MAX_N];
bool visited[MAX_N];

void dfs(int node) {
	// skip if visited
	if (visited[node]) { return; }
	visited[node] = true;

	// visit neighbors
	for (auto next : graph[node]) { dfs(next); }

	cout << node << ' ';
}

int main() {
	int n, m;
	cin >> n >> m;

	for (int i = 0; i < m; i++) {
		int u, v;
		cin >> u >> v;
		graph[u].push_back(v);
	}

	for (int i = 1; i <= n; i++) {
		if (!visited[i]) { dfs(i); }
	}

	cout << endl;
}
```
