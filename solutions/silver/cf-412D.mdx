## Explanation

Having each $a_i$ goes after its $b_i$ in the list results in a working configuration, as $b_i$ cannot directly follow $a_i$ when it is before $a_i$ in the first place.
This can always happen, except in situations where the graph formed by connecting each $a_i$ to $b_i$ has cycles.
We call each $b_i$ the neighbor of $a_i$.

Construct a directed graph with edges $a_i\to b_i$, and call $b_i$ the neighbor of $a_i$.
It is possible to make each node follow all its neighbors if there are no cycles.
This will lead to a valid configuration, as an element cannot follow another directly if it is before the latter element in the first place.
If there is a cycle, we can neglect an edge in a cycle. This is still valid, as the cycles are at least $3$ vertices.

To see why, we use an example: we require vertex A to placed before B, B to be before C, and C to be before A; from the first two requirements, A must be before C with at least $1$ element (B) in between. Since A and C are not adjacent, their order does not matter, so we may neglect the third edge.

We use a post order DFS to traverse all the nodes; at each node, we DFS all its neighbors before outputting the current node number.
This ensures that all neighbors of a vertex come before itself.
In case of a encountering a cycle while traversing, we would revisit an already-visited vertex, which we can safely skip as neglecting an edge is permissible.

We start DFS’ing from each node of it is not already visited.
Note that we may start visiting a neighbor of some other vertex we have not yet traversed; however, that other vertex would be traversed in a later DFS, so the order is still correct.  The later DFS will also skip the current neighbor, as it has already been traversed.

## Implementation

**Time complexity:** $\mathcal O(n+m)$

```cpp
// Based on solution by JuanMata on CodeForces
#include <cstring>
#include <iostream>
#include <vector>

using namespace std;

vector<int> graph[30005];
bool visited[30005];

void dfs(int node) {
	// skip if visited
	if (visited[node]) { return; }
	visited[node] = true;

	// visit neighbors
	for (auto next : graph[node]) { dfs(next); }

	cout << node << ' ';
}

int main() {
	int n, m;
	cin >> n >> m;

	for (int i = 0; i < m; i++) {
		int u, v;
		cin >> u >> v;
		graph[u].push_back(v);
	}

	memset(visited, 0, sizeof(visited));

	for (int i = 1; i <= n; i++) {
		if (!visited[i]) { dfs(i); }
	}

	cout << endl;
}
```
