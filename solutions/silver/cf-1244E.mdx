## Explanation

We want to minimize the difference between the maximum and minimum values after at most $k$ operations.

### Binary search on the answer

Instead of constructing the final array, fix a target difference $d$ and check if it is achievable.

If all elements can be moved into some interval $[L, L + d]$, then any larger $d$ also works. Hence, the condition is monotonic and we can binary search on $d$.

For a fixed $d$, after sorting, we try to fit all elements into $[L, L + d]$:
- increase elements below $L$
- decrease elements above $L + d$

The total cost can be computed using prefix sums.

### Choosing $L$
Let
```math
A = \#\{a_i < L\}, \quad B = \#\{a_i > L + d\}
```

Shifting $[L, L + d]$ (without crossing any $a_i$) changes the cost by $A - B$.
Hence, we can move the interval to improve or preserve the cost until an endpoint hits some $a_i$.

If the left endpoint hits $a_i$, then $L = a_i$.
If the right endpoint hits $a_i$, then $L + d = a_i \Rightarrow L = a_i - d$.

Thus, it suffices to consider
$L = a_i \text{ or } L = a_i - d$.

## Implementation

**Time Complexity:** $\mathcal{O}(N \log N \log A)$, where $A$ is the maximum value of $a_i$.

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	ll n, k;
	cin >> n >> k;

	vector<ll> a(n);
	for (auto &x : a) cin >> x;
	sort(a.begin(), a.end());

	vector<ll> pref(n);
	pref[0] = a[0];
	for (int i = 1; i < n; i++) pref[i] = pref[i - 1] + a[i];

	auto check = [&](ll d) -> bool {
		for (int i = 0; i < n; ++i) {
			// Case 1: L = a[i] (left endpoint pinned to a[i])
			{
				ll L = a[i], R = L + d;
				ll costLeft = (ll)i * L - (i > 0 ? pref[i - 1] : 0LL);
				int r = upper_bound(a.begin(), a.end(), R) - a.begin();
				ll costRight = r < n ? (pref[n - 1] - (r > 0 ? pref[r - 1] : 0LL)) -
				                           (ll)(n - r) * R
				                     : 0LL;
				if (costLeft + costRight <= k) return true;
			}
			// Case 2: L = a[i] - d (right endpoint pinned to a[i])
			{
				ll R = a[i], L = R - d;
				int p = lower_bound(a.begin(), a.end(), L) - a.begin();
				ll costLeft = (ll)p * L - (p > 0 ? pref[p - 1] : 0LL);
				ll costRight = (pref[n - 1] - pref[i]) - (ll)(n - i - 1) * R;
				if (costLeft + costRight <= k) return true;
			}
		}
		return false;
	};

	ll lo = 0, hi = a[n - 1] - a[0];
	while (lo < hi) {
		ll mid = lo + (hi - lo) / 2;
		if (check(mid)) hi = mid;
		else lo = mid + 1;
	}

	cout << lo << "\n";
}
```
