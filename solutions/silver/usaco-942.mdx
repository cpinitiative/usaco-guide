## Explanation

Without loss of generality, we have the left-out cow face right, and the rest facing left. Since we can flip the direction of all the cows, this covers the equivalent case of the left-out cow facing left.

Note that turning the same row or column twice has no effect.
Therefore, we can simply consider whether to turn each row and column.
We use the following procedure initially:
1. If the first cow in any row is an R, then we turn the entire row.
2. Now, the entire first column should be L.
3. If the first cow in any column is an R, then we turn the entire column.
4. Now, the entire first row should be L.

Now, we must consider the "interior region" that is neither in the first row nor the first column.
First, if there's an R in the interior region, then it is impossible to turn the entire grid to L;
attempts to get rid of that R by flipping the row or column it is contained in will result in another R in the first row or column, which means that it is impossible to get rid of all R's in this state.

In that case, there are several cases for the locations of R's in the region:
- **If there's only one R:** That R is the nonconforming cow requested by the problem.
- **If there's only a single row or column of R's:** Flip that row or column in the entire grid, and then the first cow in that row or column will be the only R, which gives us the answer.
- **If the entire region is filled with R's:**  First, flip every row besides the first row in the entire grid, and then flip the first column in the entire grid; only $(1,1)$ will be an R, which makes that our answer.
- **Any other pattern of R's:** The above cases covers single left-out cows at any location;
  therefore, in this case, there is no way to make the grid uniform by just flipping one cow.


## Implementation

**Time Complexity:  $\mathcal O(N^2)$**

```cpp
#include <fstream>
#include <vector>
using namespace std;

int main() {
	ifstream fin("leftout.in");
	ofstream fout("leftout.out");

	int n;
	fin >> n;
	vector a(n, std::vector<char>(n));
	for (int i = 0; i < n; i++) {
		for (int j = 0; j < n; j++) { fin >> a[i][j]; }
	}

	// Turn first column into L
	for (int i = 0; i < n; i++) {
		if (a[i][0] == 'R') {
			for (int j = 0; j < n; j++) { a[i][j] = 'L' + 'R' - a[i][j]; }
		}
	}

	// Turn first row into L
	for (int j = 0; j < n; j++) {
		if (a[0][j] == 'R') {
			for (int i = 0; i < n; i++) { a[i][j] = 'L' + 'R' - a[i][j]; }
		}
	}

	// Perform each of the checks
	// Interior region is all L
	bool flag = true;
	for (int i = 1; i < n; i++) {
		for (int j = 1; j < n; j++) { flag &= a[i][j] == 'L'; }
	}

	if (flag) {
		fout << -1 << endl;
		return 0;
	}

	// Interior region is all R
	flag = true;
	for (int i = 1; i < n; i++) {
		for (int j = 1; j < n; j++) { flag &= a[i][j] == 'R'; }
	}

	if (flag) {
		fout << "1 1" << endl;
		return 0;
	}

	// Singular row R in interior region
	for (int i = 1; i < n; i++) {
		flag = true;
		for (int j = 1; j < n; j++) { flag &= a[i][j] == 'R'; }

		if (flag) {
			// Verify that nothing else is R
			// If there is, put -1 as there'd be multiple that need flip
			for (int r = i + 1; r < n; r++) {
				for (int c = 1; c < n; c++) {
					if (a[r][c] == 'R') {
						fout << -1 << endl;
						return 0;
					}
				}
			}

			fout << i + 1 << ' ' << 1 << endl;
			return 0;
		}
	}

	// Singular column R in interior region
	for (int j = 1; j < n; j++) {
		flag = true;
		for (int i = 1; i < n; i++) { flag &= a[i][j] == 'R'; }

		if (flag) {
			// Verify that nothing else is R
			// If there is, put -1 as there'd be multiple that need flip
			for (int c = j + 1; c < n; c++) {
				for (int r = 1; r < n; r++) {
					if (a[r][c] == 'R') {
						fout << -1 << endl;
						return 0;
					}
				}
			}

			fout << 1 << ' ' << j + 1 << endl;
			return 0;
		}
	}

	// Single position in interior region
	int cnt = 0, saved_i, saved_j;
	for (int i = 1; i < n; i++) {
		for (int j = 1; j < n; j++) {
			if (a[i][j] == 'R') {
				cnt++;
				saved_i = i;
				saved_j = j;
			}
		}
	}

	if (cnt > 1) {
		fout << "-1" << endl;
	} else {
		fout << saved_i + 1 << ' ' << saved_j + 1 << endl;
	}
}
```
