## Implementation

**Time Complexity:** $\mathcal{O}(N)$

<PySection>

```python
with open('homework.in') as read:
        n = int(read.readline())
        questions = [int(i) for i in read.readline().split()]

left, right = -1, n
total, minimum, ans = [0] * n, [0] * n, []

total[n - 1], minimum[n - 1] = questions[n - 1], questions[n - 1]

for i in range(n - 2, 0, -1):
        cur_q = questions[i]

        total[i], minimum[i] = cur_q + total[i + 1], min(minimum[i + 1], cur_q)

        cur_s, idx = total[i] - minimum[i], (n - (i + 1))

        if cur_s * right > left * idx:
                left, right = cur_s , idx
                ans = [i]

        elif cur_s * right == left * idx:
                ans.append(i)

with open('homework.out', 'w') as out:
        for k in sorted(ans):
                print(k, file=out)
```
