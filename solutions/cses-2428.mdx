## Solution

Use sliding windows and keep a map to record the frequency of each element in the array.
Expand to the window to the right if the number of distinct elements after expanding the window is less or equal to  $\mathcal{K}$ ,
otherwise remove the left-most element.
When a new element at index $\mathcal{i}$ is added to a window with a left-most element of index $\mathcal{j}$,
it will create ($\mathcal{i - j}$) new subarrays, i.e ($\mathcal{j}$ to $\mathcal{i}$), ($\mathcal{j + 1}$ to $\mathcal{i}$), etc. which we can use to calculate the number of valid subarrays.

**Time Complexity:** $\mathcal{O}(N)$

``` java
import java.io.*;
import java.util.*;

public class cses2428 {
	public static void main(String[] args) throws IOException {
		Kattio io = new Kattio();
		int N = io.nextInt();
		int K = io.nextInt();
		int[] arr = new int[N];
		for (int i = 0; i < N; i++) {
			arr[i] = io.nextInt();
		}

		int left = 0, right = 0;
		long ans = 0;
		int distinct = 0;

		Map<Integer, Integer> freq = new HashMap<>();
		while (left < N) {
			while (right < N && (distinct + (freq.containsKey(arr[right]) ? 0 : 1) <= K)) {
				// Put the new array element in the map.
				if (!freq.containsKey(arr[right])) {
					freq.put(arr[right], 1);
					distinct++;
				} else {
					freq.put(arr[right], freq.get(arr[right]) + 1);
				}
				right++;
			}
			// Add the new subarrays.
			ans += (right - left);

			// Slide the window rightwards.
			if (freq.get(arr[left]) == 1) {
				distinct--;
				freq.remove(arr[left]);
			} else {
				freq.put(arr[left], freq.get(arr[left]) - 1);
			}
			left++;
		}

		io.println(ans);
		io.close();
	}

	// CodeSnip{Kattio}
}
```
