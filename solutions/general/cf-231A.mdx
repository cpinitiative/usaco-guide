## Explanation

The simplest solution is to input three integers per row and check
if their sum is greater than or equal to two.

## Implementation

**Time Complexity:** $\mathcal{O}(N)$

<LanguageSection>

<CPPSection>

```cpp
#include <iostream>
using namespace std;

int main() {
	int num = 0;
	int n;
	cin >> n;
	for (int i = 0; i < n; i++) {
		int a, b, c;
		cin >> a >> b >> c;
		if (a + b + c >= 2) { num++; }
	}
	cout << num << '\n';
}
```
