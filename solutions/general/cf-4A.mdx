## Explanation

We can divide the watermelon in half if and only if the weight is an even number and it's greater than 2.

## Implementation

**Time Complexity:** $\mathcal{O}(1)$

<LanguageSection>
<CPPSection>

```cpp
#include <iostream>
using namespace std;

int main() {
	int w;
	cin >> w;
	if (w % 2 == 0 && w > 2) {
		cout << "YES" << endl;
	} else {
		cout << "NO" << endl;
	}
}
```
