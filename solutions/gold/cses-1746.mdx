# Implementation


Let's define $dp[i][j]$ as the number of ways to fill the first $i$ elements of the array such that $a_i = j$.

**Base case:** if $x_1 = 0$, the first element is unknown and can be anything from $1$ to $m$, so $dp[1][j] = 1$ for all $j$. Otherwise, the first element is fixed, so $dp[1][x_1] = 1$ and all other $dp[1][j] = 0$.

For $i > 1$, since adjacent elements can differ by at most $1$, $a_i = j$ is only possible if $a_{i-1}$ was $j-1$, $j$, or $j+1$:
$$dp[i][j] = dp[i-1][j-1] + dp[i-1][j] + dp[i-1][j+1]$$

We use the same idea which we saw in [Grid Paths](https://cses.fi/problemset/task/1638), where a trap cell forces $dp[x][y] = 0$ since no path can end there, and set $dp[i][j] = 0$ whenever index $i$ is fixed to a value other than $j$, since no valid array can have $a_i = j$ in that case.

The final answer is the sum of $dp[n][j]$ over all $j$ from $1$ to $m$.

**Time Complexity:** $\mathcal{O}(NM)$

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
const int MOD = 1e9 + 7;

int main() {
	int n, m;
	cin >> n >> m;

	vector<int> a(n);
	for (int i = 0; i < n; i++) { cin >> a[i]; }

	// dp[i][j] is the number of ways to fill the first i elements
	// of the array such that the i-th element equals j
	vector<vector<ll>> dp(n + 1, vector<ll>(m + 2, 0));

	// base case: the first element is either fixed or free to be 1..m
	if (a[0] == 0) {
		for (int j = 1; j <= m; j++) { dp[1][j] = 1; }
	} else {
		dp[1][a[0]] = 1;
	}

	for (int i = 2; i <= n; i++) {
		int curr = a[i - 1];
		for (int j = 1; j <= m; j++) {
			// if this position is fixed and doesn't match j, it's unreachable
			if (curr != 0 && curr != j) continue;

			// otherwise, j can be reached from j-1, j, or j+1 at the previous index
			dp[i][j] = dp[i - 1][j];
			if (j - 1 >= 1) dp[i][j] += dp[i - 1][j - 1];
			if (j + 1 <= m) dp[i][j] += dp[i - 1][j + 1];
			dp[i][j] %= MOD;
		}
	}

	ll ans = 0;
	for (int j = 1; j <= m; j++) { ans = (ans + dp[n][j]) % MOD; }

	cout << ans << '\n';
}
```

<JavaSection>

```java

import java.io.*;
import java.util.*;

public class ArrayDesc {
	public static final int MOD = (int)1e9 + 7;
	public static void main(String args[]) throws IOException {
		BufferedReader r = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(r.readLine());
		int N = Integer.parseInt(st.nextToken());
		int M = Integer.parseInt(st.nextToken());

		// DP[i][j] is the number of ways
		// to make a valid array from 0 to i, if the ith value is j.
		int[][] dp = new int[N][M + 1];

		// Process the first element in the array.
		StringTokenizer arr = new StringTokenizer(r.readLine());
		int first = Integer.parseInt(arr.nextToken());
		// The first element can be anything.
		if (first == 0) {
			Arrays.fill(dp[0], 1);
		} else {
			// This is the only valid value.
			dp[0][first] = 1;
		}

		for (int i = 1; i < N; i++) {
			int A = Integer.parseInt(arr.nextToken());
			// A free element.
			if (A == 0) {
				for (int j = 1; j <= M; j++) {
					int[] neighborVals = new int[] {j - 1, j, j + 1};
					for (int k : neighborVals) {
						// Number of values,
						// where the previous value is in the valid range.
						if (1 <= k && k <= M) {
							dp[i][j] += dp[i - 1][k];
							dp[i][j] %= MOD;
						}
					}
				}
			} else {
				// Find number of ways that the current value is valid.
				int[] neighborVals = new int[] {A - 1, A, A + 1};
				for (int k : neighborVals) {
					if (1 <= k && k <= M) {
						dp[i][A] += dp[i - 1][k];
						dp[i][A] %= MOD;
					}
				}
			}
		}

		// Aggregate the dp values.
		int ans = 0;
		for (int j = 1; j <= M; j++) {
			ans += dp[N - 1][j];
			ans %= MOD;
		}
		System.out.println(ans);
	}
}
```
