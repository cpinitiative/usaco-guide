## Explanation


Let's define $dp[i][j]$ as the number of ways to fill the first $i$ elements of the array, such that $a_i = j$.

Base case: if $x_1 = 0$, then the first element is unknown and can be anything from $1$ to $m$, so $dp[1][j] = 1$ for all $j$. Otherwise, the first element is fixed, so $dp[1][x_1] = 1$ and all other $dp[1][j] = 0$.

For $i > 1$, since adjacent elements differ by at most $1$, $a_i = j$ is possibly only if $a_{i-1}$ is $j-1$, $j$, or $j+1$:

$$
dp[i][j] = dp[i-1][j-1] + dp[i-1][j] + dp[i-1][j+1]
$$

We can use the same idea seen in [Grid Paths](https://cses.fi/problemset/task/1638), where a trap cell forces $dp[x][y] = 0$ since no path can end there. Then, we set $dp[i][j] = 0$ whenever index $i$ is fixed to a value other than $j$, since no valid array can have $a_i = j$ in that case.

We compute the final answer by summing $dp[n][j]$ over all $j$ from $1$ to $m$.

## Implementation

**Time Complexity:** $\mathcal{O}(NM)$

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
const int MOD = 1e9 + 7;

int main() {
	int n, m;
	cin >> n >> m;

	vector<int> values(n + 1);
	for (int i = 1; i <= n; i++) { cin >> values[i]; }

	// dp[position][value] is the number of ways to fill the first "position" elements
	// of the array such that the element at "position" equals value
	vector<vector<ll>> dp(n + 1, vector<ll>(m + 1));

	// base case: the first element is either fixed or free to be 1..m
	if (values[1] == 0) {
		for (int j = 1; j <= m; j++) { dp[1][j] = 1; }
	} else {
		dp[1][values[1]] = 1;
	}

	for (int position = 2; position <= n; position++) {
		int fixed_value = values[position];
		for (int val = 1; val <= m; val++) {
			// if this position is fixed and doesn't match j, it's unreachable
			if (fixed_value != 0 && fixed_value != val) continue;

			// otherwise, j can be reached from j-1, j, or j+1 at the previous index
			dp[position][val] = dp[position - 1][val];
			if (val - 1 >= 1) dp[position][val] += dp[position - 1][val - 1];
			if (val + 1 <= m) dp[position][val] += dp[position - 1][val + 1];
			dp[position][val] %= MOD;
		}
	}

	ll ans = 0;
	for (int j = 1; j <= m; j++) { ans = (ans + dp[n][j]) % MOD; }

	cout << ans << '\n';
}
```

```java

import java.io.*;
import java.util.*;

public class ArrayDesc {
	public static final int MOD = (int)1e9 + 7;
	public static void main(String args[]) throws IOException {
		BufferedReader r = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(r.readLine());
		int N = Integer.parseInt(st.nextToken());
		int M = Integer.parseInt(st.nextToken());

		// dp[i][j] is the number of ways to fill the first i+1 elements
		// of the array such that the i-th element equals j
		int[][] dp = new int[N][M + 1];

		// base case: the first element is either fixed or free to be 1..M
		StringTokenizer arr = new StringTokenizer(r.readLine());
		int first = Integer.parseInt(arr.nextToken());

		// if the first element is free, it may be any value from 1..M
		if (first == 0) {
			Arrays.fill(dp[0], 1);
		} else {
			// otherwise, the first element is fixed
			dp[0][first] = 1;
		}

		for (int i = 1; i < N; i++) {
			int A = Integer.parseInt(arr.nextToken());
			// if this position is free, try all values 1..M
			if (A == 0) {
				for (int j = 1; j <= M; j++) {
					int[] neighborVals = new int[] {j - 1, j, j + 1};
					for (int k : neighborVals) {
						// j can be reached from j-1, j, or j+1 at the previous index
						if (1 <= k && k <= M) {
							dp[i][j] += dp[i - 1][k];
							dp[i][j] %= MOD;
						}
					}
				}
			} else {
				// if this position is fixed to A,
				// it can only be reached from A-1, A, or A+1
				int[] neighborVals = new int[] {A - 1, A, A + 1};
				for (int k : neighborVals) {
					if (1 <= k && k <= M) {
						dp[i][A] += dp[i - 1][k];
						dp[i][A] %= MOD;
					}
				}
			}
		}

		// sum over all possible values of the last element
		int ans = 0;
		for (int j = 1; j <= M; j++) {
			ans += dp[N - 1][j];
			ans %= MOD;
		}

		System.out.println(ans);
	}
}
```
