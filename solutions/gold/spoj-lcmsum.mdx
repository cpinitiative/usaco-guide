## Explanation

We'll use the following identity in trying to bring the given sum to a more convenient form:
$$
x \cdot y=\gcd(x, y)\cdot\text{lcm}(x, y)
$$

$$
\sum_{i = 1}^{n}{\texttt{lcm}(i, n)} = \sum_{i = 1}^{n}{\frac{i \cdot n}{\gcd(i, n)}}=n\cdot\sum_{i=1}^{n}{\frac{i}{\gcd(i, n)}}
$$

We try grouping the terms by their value. Let $d=\gcd(i, n)$, then $d|n$:

$$
n\cdot\sum_{i=1}^{n}{\frac{i}{\gcd(i, n)}}=n \cdot \sum_{d|n}\sum_{\substack{i=1\\gcd(i,n)=d}}^{n}{\frac{i}{d}} \tag{1}
$$

Let $k = \frac{i}{d}$ and $m= \frac{n}{d}$, from this results that $\gcd(k, m)=1$. Substituing back into $(1)$ we get:

$$
n\cdot\sum_{d|n}\sum_{\substack{k=1 \\ \text{gcd(k,m)=1}}}^{m}{k}
$$


The inner sum is equal to $\frac{m \cdot \phi(m)}{2}$ for $m > 1$.

## Implementation

**Time Complexity:** $\mathcal{O}(N\log{N} + T)$

```cpp
#include <iostream>

using namespace std;

const int MAXN = 1e6;

long long phi[MAXN + 1], sum[MAXN + 1];

int main() {

	ios_base::sync_with_stdio(false);
	cin.tie(nullptr);

	for (int i = 1; i <= MAXN; i++) { phi[i] = i; }

	for (int i = 2; i <= MAXN; i++) {
		// If i is prime
		if (phi[i] == i) {
			for (int j = i; j <= MAXN; j += i) { phi[j] -= phi[j] / i; }
		}
	}

	for (int i = 1; i <= MAXN; i++) {
		for (int j = i; j <= MAXN; j += i) { sum[j] += i * phi[i]; }
	}

	int t;
	cin >> t;
	
	while (t--) {
		int n;
		cin >> n;
		long long ans = sum[n] + 1;
		ans = 1LL * ans * n / 2LL;
		cout << ans << '\n';
	}

	return 0;
}
```
