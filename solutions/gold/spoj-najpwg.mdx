## Explanation

The queries can be answered in $\mathcal{O}(1)$ by performing prefix sums of $f(n)$, where $f(n) = n - \phi(n)$. The reason we take the difference is
that the $\phi(n)$ counts number of values $x \le n$ with $(x, n)=1$, but we want the pairs with $(x,n) > 1$ which are the rest.

## Implementation

**Time Complexity:** $\mathcal{O}(N\log\log N + N)$

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5;

int phi[MAXN + 1];

void precompute() {
	for (int i = 1; i <= MAXN; i++) { phi[i] = i; }

	for (int i = 2; i <= MAXN; i++) {
		// If i is prime
		if (phi[i] == i) {
			for (int j = i; j <= MAXN; j += i) { phi[j] -= phi[j] / i; }
		}
	}

	for (int i = 1; i <= MAXN; i++) {
		// Add the pairs with gcd > 1 which is i - phi[i]
		phi[i] = phi[i - 1] + (i - phi[i]);
	}
}

int main() {
	int t;
	cin >> t;

	precompute();

	for (int i = 1; i <= t; i++) {
		int n;
		cin >> n;
		cout << "Case " << i << ": " << phi[n] << '\n';
	}
}
```
