## Explanation
We can begin by noting that all ranges that are removed must be disjoint. If two
ranges overlap, they can be combined into one segment. If one
segment is within another, then the largest segment can replace the smaller
segment.

Using this idea, the goal can be thought of as the minimum number of balls
remaining. We define $dp[i]$ as the minimum number of balls left after looking at the first $i$ balls.

At ball $i$, we have two options:
* Don’t use it in any segment, so $dp[i] = dp[i-1] + 1$
* Pair it with an earlier ball of the same color at position $k$ and delete all balls from $k$ to $i$, so $dp[i] = dp[k-1]$

The formula can be further condensed into

$$
dp[i] = \min(dp[i-1] + 1,\ \texttt{best}[a_i])
$$

where $\texttt{best}[a_i]$ stores the minimum value of $dp[k-1]$ over all previous positions $k$ such that $a_k = a_i$.

After finding $dp[i]$, we update $\texttt{best}[a_i] = \min(\texttt{best}[a_i], dp[i-1])$.

The final answer is $N - dp[N]$.

## Solution

<LanguageSection>

<PySection>

**Time Complexity:** $\mathcal{O}(N\log{N})$

```py
INF = 10**20

T = int(input())
for _ in range(T):
	N = int(input())
	balls = list(map(int, input().split()))
	dp = [INF] * (N + 1)
	dp[0] = 0

	best = [INF] * (N + 1)
	for i in range(1, N + 1):
		# keep or delete segment ending at index i
		dp[i] = min(dp[i - 1] + 1, best[balls[i - 1]])
		# update best for this color with dp[i-1]
		best[balls[i - 1]] = min(best[balls[i - 1]], dp[i - 1])

	print(N - dp[N])
```

<CPPSection>

**Time Complexity:** $\mathcal{O}(N)$

```cpp
#include <bits/stdc++.h>
using namespace std;

void solve() {
	int n;
	cin >> n;
	vector<int> v(n);
	for (int i = 0; i < n; i++) { cin >> v[i]; }

	vector<int> dp(n + 1, 1e9);
	vector<int> best(n + 1, 1e9);
	dp[0] = 0;

	for (int i = 1; i <= n; i++) {
		dp[i] = min(dp[i - 1] + 1, best[v[i - 1]]);
		best[v[i - 1]] = min(best[v[i - 1]], dp[i - 1]);
	}

	cout << (n - dp[n]) << "\n";
}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(NULL);
	int t;
	cin >> t;
	while (t--) { solve(); }
	return 0;
}
```
