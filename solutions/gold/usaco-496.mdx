## Explanation

This problem wants the minimum number of lanes (longest increasing subsequences) such that no cows overlap during running.

Since the input is already sorted by $p$, notice that if two cows have $$p_{i}+T \cdot s_{i} >= p_{i+1}+T \cdot s_{i+1}$$ they must have passed each other along the way.

We want to ensure the opposite within each lane: $$p_{i}+T \cdot s_{i} < p_{i+1}+T \cdot s_{i+1}$$
So we count longest strictly increasing subsequences of $p+T \cdot s$ which is the same as the length of the longest non-increasing subsequence.

**Time Complexity:** $\matchcal{O}(N\logN)$

#include <bits/stdc++.h>
using namespace std;
int main()
{
	freopen("cowjog.in","r",stdin);
	freopen("cowjog.out","w",stdout);
	int n, t;
	cin>>n>>t;
	uint_fast64_t lds[100'000];
	int *last=lds;
	for(int i=0;i<n;++i){//finds the min # of strictly increasing subsequences
		uint_fast64_t pos, spd;
		cin>>pos>>spd;
		pos+=spd*t;//1 billion*1 billion
		auto it = upper_bound(lds,last,pos,greater{});
		*it = pos;
		last += (it==last);
	}
	cout<<last-lds<<'\n';
}
