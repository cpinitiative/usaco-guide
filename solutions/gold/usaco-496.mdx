## Explanation

We are asked to calculate the minimum number of lanes (longest increasing subsequences), such that no cows overlap while running.

Since the input is already sorted by $p$, notice that if two cows have $p_{i}+T \cdot s_{i} \geq p_{i+1}+T \cdot s_{i+1}$, they must have passed each other along the way.

We want to ensure the opposite within each lane: $p_{i}+T \cdot s_{i} < p_{i+1}+T \cdot s_{i+1}$, so we calculate the minimum longest strictly increasing subsequences of $p+T \cdot s$.
This is equivalent to the length of the longest non-increasing subsequence, as mentioned [in the module](/gold/lis#application-2---minimum-number-of-increasing-sequences).

## Implementation

**Time Complexity:** $\mathcal{O}(N \log N)$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	freopen("cowjog.in", "r", stdin);
	freopen("cowjog.out", "w", stdout);

	int n, t;
	cin >> n >> t;
	vector<long long> lds;

	// find min # of strictly increasing subsequences
	for (int i = 0; i < n; ++i) {  
		long long pos, spd;
		cin >> pos >> spd;
		pos += spd * t;
		auto it = upper_bound(lds.begin(), lds.end(), pos, greater{});
		if (it == lds.end()) {
			lds.push_back(pos);
		} else {
			*it = pos;
		}
	}

	cout << lds.size() << '\n';
}
```
