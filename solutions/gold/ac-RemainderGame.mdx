## Explanation

Applying any modulus twice is identical to applying it only once, even when other moduli are used, so we should purchase each modulus at most once. Observe that a large modulus always costs more than any combination of smaller moduli, since the worst case is purchasing every smaller modulus once:

$$
2^k > \sum_{i=1}^{k-1} 2^i.
$$

This means we should prioritize optimizing the largest modulus used across all transformations $a_i \to b_i$, which motivates our greedy idea: repeatedly find the largest modulus we must buy, and buy it exactly once.

To do this we can build a directed graph on vertices $0, \ldots, 50$ where for every value $x$ and modulus $k$, there is an edge $x \to x \bmod k$ with weight $k$. A path from $a_i$ to $b_i$ represents a sequence of operations turning $a_i$ into $b_i$, for which the moduli needed are the edge weights along the path. Define the weight of a path from $u$ to $v$, $f(u, v)$, as the minimum possible value of the maximum edge weight on such a path.

Hence, when processing the transformation $a_i \to b_i$, $f(a_i, b_i)$ is the minimum threshold such that the path is feasible using only operations $\leq f(a_i,b_i)$, making $K = \max_i f(a_i, b_i)$ a forced modulus in the greedy construction.

We compute these minimax distances with a Floyd–Warshall variant, replacing the standard relaxation with:

$$\texttt{dist}[i][j] = \min(\texttt{dist}[i][j],\ \max(\texttt{dist}[i][k],\ \texttt{dist}[k][j])).$$

Initially, `dist` holds the direct graph edges. We run minimax Floyd–Warshall to get all $f(u, v)$ given the current edge weights, then take $K = \max_i f(a_i, b_i)$, the largest modulus some pair still requires. If any pair is unreachable, output `-1`. If $K = 0$, every necessary modulus is already purchased, so we stop. Otherwise we add $2^K$ to the answer, reset `dist` to direct edges, and set every edge of weight $K$ to $0$, marking modulus $K$ as purchased. We repeat until done.

Each iteration permanently purchases one modulus, so there are at most $50$
iterations.

## Implementation

Let $M = 50$ be the maximum value involved in the problem. Each Floyd–Warshall pass is $\mathcal{O}(M^3)$ on $M + 1$ vertices, repeated at most $M$ times, giving an
overall complexity of $\mathcal{O}(M^4)$.

We keep `edges_by_modulus[k]`, a list of every edge with weight $k$, so that once modulus $k$ is purchased, we can mark edges using that modulus as free in $\mathcal{O}(1)$ per edge, without scanning the whole matrix.

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAX_VAL = 50;
const int NUM_NODES = MAX_VAL + 1;
const int INF = 1e9;

int dist[NUM_NODES][NUM_NODES];
int temp_dist[NUM_NODES][NUM_NODES];

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	int n;
	cin >> n;

	vector<int> a(n), b(n);
	for (int &x : a) cin >> x;
	for (int &x : b) cin >> x;

	vector<pair<int, int>> edges_by_modulus[NUM_NODES];

	for (int i = 0; i <= MAX_VAL; i++) {
		for (int j = 0; j <= MAX_VAL; j++) { dist[i][j] = INF; }
		dist[i][i] = 0;
	}

	// Add an edge x -> x % k with weight k, store for fast lookup later
	for (int x = 0; x <= MAX_VAL; x++) {
		for (int k = 1; k <= MAX_VAL; k++) {
			int y = x % k;
			dist[x][y] = min(dist[x][y], k);
			edges_by_modulus[k].push_back({x, y});
		}
	}

	long long ans = 0;

	while (true) {
		memcpy(temp_dist, dist, sizeof(dist));

		// Minimax Floyd-Warshall under the current (partially purchased) weights
		for (int mid = 0; mid <= MAX_VAL; mid++) {
			for (int i = 0; i <= MAX_VAL; i++) {
				for (int j = 0; j <= MAX_VAL; j++) {
					temp_dist[i][j] =
					    min(temp_dist[i][j], max(temp_dist[i][mid], temp_dist[mid][j]));
				}
			}
		}

		int largest_required = 0;
		for (int i = 0; i < n; i++) {
			if (temp_dist[a[i]][b[i]] == INF) {
				cout << -1 << "\n";
				return 0;
			}
			largest_required = max(largest_required, temp_dist[a[i]][b[i]]);
		}

		if (largest_required == 0) break;

		ans += (1LL << largest_required);

		// Mark every edge using this modulus as free for future iterations
		for (auto [from, to] : edges_by_modulus[largest_required]) {
			dist[from][to] = 0;
		}
	}

	cout << ans << "\n";
}
```
