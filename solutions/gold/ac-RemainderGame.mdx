## Explanation
Since modulo can never increase a value, if $a_i < b_i$ for some $i$, the answer is immediately $-1$.

Instead of tracking operations, let's track which **moduli** we use. Using modulus $k$ costs $2^k$, and lets us reduce any element mod $k$

Applying any modulus twice is identical to applying it only once, so we should purchase each modulus at most once.
Now, note that a large modulus always costs more than any combination of smaller moduli, as the worst combination is to purchase each moduli only once, and we have:

$$2^k > \sum_{i=1}^{k-1} 2^i.$$

This means that we should prioritize optimizing the largest modulus used in all transformations $a_i \to b_i$ each time, which motivates a greedy idea:
repeatedly finding the largest modulus every solution is forced to buy, and buying that modulus exactly once.

Let's build a directed graph on vertices $0, \ldots, 50$. For every value $x$ and modulus $k$, add an edge

$$x \to x \bmod k$$

with weight $k$. A path from $a_i$ to $b_i$ represents a sequence of operations which turn $a_i$ into $b_i$, and the moduli needed are the edge weights on that path. Define a *path's* weight as its **maximum** edge weight, and let

$$f(u, v) = \min_{\text{paths } u \to v} \ \max(\text{edge weight}).$$

Hence, when processing moduli for the path $a_i \to b_i$, $f(a_i, b_i) = K$ means $K$ is the minimum threshold such that the path becomes feasible using only operations $\le K$, so in the across all the paths this is the first point where this pair cannot be satisfied without introducing modulo $K$, making it a forced contribution in the greedy construction.

We can compute these minimax distances with a Floyd–Warshall variant:

```cpp
dist[i][j] = min(dist[i][j], max(dist[i][k], dist[k][j]));
```

Initially `dist` holds the direct graph edges, and we repeat the following until done. We run minimax Floyd–Warshall to get all $f(u, v)$ given the current edge
weights, then take $K = \max_i f(a_i, b_i)$, which is the largest modulus some pair still requires. If any pair is unreachable, output `-1`. If $K = 0$, every necessary modulus is already purchased, so we stop. Otherwise we add $2^K$ to the answer and set every edge of weight $K$ to $0$, marking modulus $K$ as purchased and free for all future paths.

Each iteration permanently uses one modulus, so there are at most $50$ iterations.

## Implementation

Each Floyd–Warshall pass is $\mathcal{O}(51^3)$ on $51$ vertices, repeated at most
$50$ times, giving an overall complexity of

$$\mathcal{O}(50 \cdot 51^3) = \mathcal{O}(50^4).$$

We keep `edges_by_modulus[k]`, a list of every edge with weight $k$, so that once
modulus $k$ is purchased we can mark exactly those edges as free in $\mathcal{O}(1)$
per edge, without scanning the whole adjacency matrix.

```cpp
#include <bits/stdc++.h>
using namespace std;

const int num_values = 51;
const int inf = 1e9;

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	int n;
	cin >> n;

	vector<int> a(n), b(n);
	for (int &x : a) cin >> x;
	for (int &x : b) cin >> x;

	int dist[num_values][num_values];
	vector<pair<int, int>> edges_by_modulus[num_values];

	for (int i = 0; i < num_values; i++) {
		for (int j = 0; j < num_values; j++) dist[i][j] = inf;
		dist[i][i] = 0;
	}

	// Add an edge x -> x % k with weight k, and store it for fast lookup later
	for (int x = 0; x < num_values; x++) {
		for (int k = 1; k <= 50; k++) {
			int y = x % k;
			dist[x][y] = min(dist[x][y], k);
			edges_by_modulus[k].push_back({x, y});
		}
	}

	long long ans = 0;

	while (true) {
		int temp_dist[num_values][num_values];
		memcpy(temp_dist, dist, sizeof(dist));

		// Minimax Floyd-Warshall under the current (partially purchased) weights
		for (int mid = 0; mid < num_values; mid++)
			for (int i = 0; i < num_values; i++)
				for (int j = 0; j < num_values; j++)
					temp_dist[i][j] =
					    min(temp_dist[i][j], max(temp_dist[i][mid], temp_dist[mid][j]));

		int largest_required = 0;
		for (int i = 0; i < n; i++) {
			if (temp_dist[a[i]][b[i]] == inf) {
				cout << -1 << "\n";
				return 0;
			}
			largest_required = max(largest_required, temp_dist[a[i]][b[i]]);
		}

		if (largest_required == 0) break;

		ans += (1LL << largest_required);

		// Mark every edge using this modulus as free for future iterations
		for (auto [from, to] : edges_by_modulus[largest_required]) {
			dist[from][to] = 0;
		}
	}

	cout << ans << "\n";
}
```
