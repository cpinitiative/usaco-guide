## Solution

In this problem, we need to find the maximum amount of money we can earn by attending Projects. 
This can be solved by firstly sorting every perojects by their ending times and then using set of base case zero. Use lower_bound function to find the enterval of ending time less then starting time and inserting the ending time into the set to get the last maximum value. 

Also check [Unofficial Editorial](https://codeforces.com/blog/entry/70018).

```cpp

#include <bits/stdc++.h>

using namespace std;

#define ll long long


const int MXN=2e5+10;
ll n;
array<ll, 3> a[MXN];

int main(){
    cin >> n;
    for(ll i = 0; i < n; i++){
        cin >> a[i][1] >> a[i][0] >>a[i][2];
    }
    sort(a, a+n); // sort everyone by their ending times.
    set<array<ll, 2>> s;
    s.insert({0, 0}); // base case zero
    ll ans = 0;
    for(ll i = 0; i < n; i++){
        auto it = s.lower_bound({a[i][1]});
        --it;
        ans = max(ans, (*it)[1]+a[i][2]);
        s.insert({a[i][0], ans}); // inserting the ending time into the set
    }
    cout << ans;
}
```
