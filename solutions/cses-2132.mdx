**Time Complexity**: $\mathcal O(N\log N)$

This problem is similar to
[CF 713C](https://codeforces.com/contest/713/problem/C) except that we're
supposed to create a **non-decreasing array**.

Here is [zscoder's tutorial](https://codeforces.com/blog/entry/47821).

<CPPSection>

```cpp
#include <stdio.h>
#include <queue>
using namespace std;

int n, t;
long long ans = 0;
priority_queue<int> Q;

int main() {
	scanf("%d", &n);
	for(int i=0; i<n; i++) {
		scanf("%d", &t);
		Q.push(t);
		ans += Q.top() - t;
		Q.pop(); Q.push(t);
	}
	printf("%lld", ans);
	return 0;
}
```

import java.util.*;

class IncreasingSeq {

    public static void main(String args[]) {

        PriorityQueue < Integer > pQueue = new PriorityQueue < Integer > (
            Collections.reverseOrder());
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();

        int[] arr;
        arr = new int[n];
        long ans = 0;
        for (int i = 0; i < n; i++) {
            int k = sc.nextInt();
            pQueue.add(k);
            int pp = pQueue.peek();
            ans += pp - k;
            pQueue.poll();
            pQueue.add(k);
        }
        System.out.println(ans);
    }
}
