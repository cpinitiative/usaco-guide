## Explanation

If 
- `s` is the string representing the text from the magazine
-  `t` is the string representing the censored word
- `ans` is the string representing the answer
 then...

We can simply iterate through every character of the string `s` , appending it to `ans`. Then, whenever we append a character we can try to see if `ans` ends with the censored word (i.e the last `t.size()` characters of `ans` make up `t`). If this is so, we can remove it from `ans` (i.e remove the last `t.size()` characters from `ans`).

This method of looping through every character and looking back to see if we've written a censored word allows us to tackle test cases where `s` contains censored words within censored words such as in the example given in the question, where ...

`s` = `"whatthemomooofun"`\
`t` = `"moo"`

The program will loop through each character of `s`, appending it to `ans` as normal untill `ans` becomes `"whatthemomoo"`. At this point it will omit the last 3 characters of `ans` as they form the censored word. Hence, `ans` becomes `"whatthemo"`. After the next iteration `ans` becomes `"whatthemoo"` because the next letter in `s` is an 'o'. Again the last 3 words are omitted and `ans` becomes `"whatthe"`. The loop will continue normally and `ans` will end up as `"whatthefun"` as intended.

## Implementation

**Time Complexity:** $\mathcal{O}(ST)$

```py
import sys
sys.stdin = open("censor.in", "r")
sys.stdout = open("censor.out", "w")

s = input().strip()
t = input().strip()
final = ""
# Add each character to the final string
for char in s:
	final += char
	# If the end of the string is t, we remove t from the end
	if final[-len(t):] == t:
		final = final[:-len(t)]
print(final)
```

```cpp
int main() {
  setIO("censor");
  string s,t,ans; cin >> s >> t;
  for(int i=0;i<s.size();i++){
    ans += s[i];
	if (ans.size()>t.size() && ans.substr(ans.size()-t.size()) == t)
		ans.resize(ans.size() - t.size());  //If ans ends in the string t, we remove t from the end
  } cout << ans;
}
```

```java
import java.util.*;
import java.io.*;

public class Censor {
	public static void main (String[] args) throws IOException {
		Kattio io = new Kattio("censor");
		String string = io.next();
		String toCensor = io.next();

		// use StringBuilder in Java because it is significantly faster
		StringBuilder s = new StringBuilder();
		s.append(string.substring(0, toCensor.length() - 1));

		// try adding one character at a time
		for (int x = toCensor.length() - 1; x < string.length(); x++) {
			s = s.append(string.charAt(x));
			// need to check if our current string is longer than the censored word
			if (s.length() >= toCensor.length()) {
				String check = s.substring(s.length() - toCensor.length());
				// delete the censored word if present
				if (check.equals(toCensor)) {
					s.delete(s.length() - toCensor.length(), s.length());
				}
			}
		}
		io.println(s);
		io.close();
	}
	//CodeSnip{Kattio}
}
```
