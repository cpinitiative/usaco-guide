# Solution


**Time Complexity:** $\mathcal{O}(N^2)$

This problem is an extension of the two sum problem except now with three values. We can set the third pointer to a certain value in the array and then the problem becomes the two sum problem discussed earlier in this module. We'll first sort the array, then we'll loop through all possible values for the third pointer and check if all three pointers add up to out target amount, $x$, while having distinct positions. We can maintain the original positions of all values by storing them as a pair with the value in the array being the first value and the position being in the second value.


```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(0); cin.tie(0);
    int n,x; cin >> n >> x;
    vector<pair<int, int>> arr;
    for(int i = 1; i <= n; i++){
        int a; cin >> a;
        pair<int, int> p; p.first = a; p.second = i; //first in pair represents value, second represents index
        arr.push_back(p);
    }
    sort(begin(arr), end(arr));
    for(int i = 0; i < n; i++){
        int l, r; l = 0; r = n-1;
        while(l != r){
            int target; target = x - arr.at(i).first;
            if(l != i && r != i && arr.at(l).first + arr.at(r).first == target){
                cout << arr.at(i).second << " " << arr.at(l).second << " " << arr.at(r).second << endl;
                return 0;
            }
            if(arr.at(l).first + arr.at(r).first < target){
                l++;
            }
            else{
                r--;
            }
        }
    }
    cout << "IMPOSSIBLE" << endl;
}
```
