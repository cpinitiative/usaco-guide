## Solution

[Unofficial Editorial](https://codeforces.com/blog/entry/70018).

**Time Complexity:** $\mathcal{O}(n)$


```cpp

#include <bits/stdc++.h>
#include <iostream>
using namespace std;


int main()
{
    int mod = 1e9+7;
    long n;
    cin >> n;

    int k[7] = {1};
    for (int i = 1; i <= n; ++i) {
    	rotate(k, k+6, k+7);
    	k[0] = 0;
    	for(int j = 1; j < 7; ++j)
    	{
            k[0] += k[j];
            k[0] %= mod;
    	}
    }
    cout << k[0] << "\n";
    return 0;
}
```
