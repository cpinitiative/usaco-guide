## Explanation

We can say that a cow crossed the road if its current side is
different from its previous side. Based on this, we will check 
each of Farmer John's cows' current side and previous side (if any), 
and if there is any difference we will increment our number of 
confirmed crossings.

## Implementation

**Time Complexity:** $\mathcal{O}(N)$

```cpp
#include <iostream>
#include <string>
#include <vector>

void set_up(std::string name) {
    freopen((name + ".in").c_str(), "r", stdin);
    freopen((name + ".out").c_str(), "w", stdout);
}

int main() {
    set_up("crossroad");
    int n;
    std::cin >> n;
    std::vector<int> cows(10, -1);
    
    int crossings = 0;

    for (int i = 1; i <= n; i++) {
        int cow_id, side;
        std::cin >> cow_id >> side;
        cow_id--;

        if (cows[cow_id] == -1) {
            cows[cow_id] = side;
        } else if (cows[cow_id] != side) {
            crossings++;
            cows[cow_id] = side;
        }
    }
    
    std::cout << crossings << '\n';
}
```

```py
data = open("crossroad.in").read().strip().split("\n")
n = int(data[0])

observations = [list(map(int, data[i + 1].split(" "))) for i in range(n)]

locations = [""] * 10

crossings = 0

for i in range(n):
	if locations[observations[i][0] - 1] == "":
		locations[observations[i][0] - 1] = observations[i][1]
	elif locations[observations[i][0] - 1] != observations[i][1]:
		locations[observations[i][0] - 1] = observations[i][1]
		crossings += 1

print(crossings, file=open("crossroad.out", "w"))
```
