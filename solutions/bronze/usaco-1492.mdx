## Explanation

To make the mex of the array equal $i$, the array must contain all elements from $0$ to $i - 1$, and there must be no occurrences of $i$. Let the number of missing elements in the array from $0$ to $i - 1$ be $x$ and the number of occurrences of $i$ be $y$. Our goal is to make $x$ and $y$ zero using the operation.

If $y \ge x$, we can change $x$ elements equal to $i$ to fill in the $x$ missing spots, and the $y - x$ remaining elements equal to $i$ to anything not equal to $i$, for a total of $y$ operations. If $y < x$, we can change $y$ elements equal to $i$ to fill in $y$ missing spots and then change $x - y$ more elements to fill in the remaining $x - y$ remaining missing spots. The answer is thus $\max(x, y)$. These quantities can be found by using a frequency array or map.

This is optimal because in each operation, we decrease both $x$ and $y$ by one for each operation if they are nonzero, or the nonzero quantity if one is zero.

## Implementation

**Time Complexity:** $\mathcal{O}(N)$

<CPPSection>

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int n;
	cin >> n;

	vector<int> array(n);
	for (int i = 0; i < n; i++) { cin >> array[i]; }

	vector<int> freq(n + 1, 0);
	int missing = 0;

	for (int x : array) { freq[x]++; }

	for (int i = 0; i <= n; i++) {
		cout << max(missing, freq[i]) << endl;
		if (freq[i] == 0) { missing++; }
	}
}
```

<PySection>

```py
n = int(input())
array = [int(x) for x in input().split()]
freq = [0] * (n + 1)
missing = 0

for x in array:
	freq[x] += 1

for i in range(n + 1):
	print(max(missing, freq[i]))
	if freq[i] == 0:
		missing += 1
```
