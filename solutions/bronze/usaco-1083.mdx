## Explanation

We know that Bessie hums the cowphabet in a fixed order. Based on
this, we iterate over consecutive pairs of letters in Farmer
John's word. If the current letter appears before or at the same
position as the next letter in the cowphabet, there is no need for
Bessie to hum; otherwise, we increment $hum\_times$.

## Implementation

**Time Complexity:** $\mathcal{O}(|\texttt{word}|)$

<LanguageSection>
<CPPSection>

```cpp
#include <iostream>
#include <string>

int main() {
    std::string cowphabet, word;
    std::cin >> cowphabet >> word;

    int hum_times = 1; // There must be at least one humming sequence.
    for (int i = 0; i < (int)word.size() - 1; i++) {
        /*
         * Increment if the next letter appears at
         * or earlier than the current letter in the alphabet
         */
        if (cowphabet.find(word[i + 1]) <= cowphabet.find(word[i])) {
            hum_times++;
        }
    }

    std::cout << hum_times << '\n';

}
```

# Take in the cowphabet and string as strings through standard Python input
cowphabet = input()

	# Check if next character in string appears before previous character

<JavaSection>

```java
import java.io.*;

public class UdderedButNotHerd {
	public static void main(String[] args) throws IOException {
		BufferedReader input = new BufferedReader(new InputStreamReader(System.in));
		String cowphabet = input.readLine();
		char[] word = input.readLine().toCharArray();

		int humTimes = 1;  // There must be at least one humming sequence.
		for (int i = 0; i < word.length - 1; i++) {
			/*
			 * Increment if the next letter appears at
			 * or earlier than the current letter in the alphabet
			 */
			if (cowphabet.indexOf(word[i + 1]) <= cowphabet.indexOf(word[i])) {
				humTimes++;
			}
		}
		System.out.println(humTimes);
	}
}
```
