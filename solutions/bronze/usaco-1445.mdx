## Explanation

Notice that modifying exactly one character can only affect at most $3$ substrings.

For example, had the string been $\text{ABCDEF}$ and the $\text{C}$ was modified to a $\text{Z}$, the affected substrings would now be, $\text{ABZ}$, $\text{BZD}$, and $\text{ZDE}$.

Using this, we first precompute a map representing the counts of each moo that was originally present in the string. Then, we can try every single possible edit to the string, with a total of $\mathcal{O}(26N)$ possible modifications. While trying every single possible edit to a specific position, due to only three substrings being affected, the global answer does not change by much. So, we can adjust the frequencies of moo's based on the $3$ substrings being affected after every modification, and check whether any moo's exceeded a frequency of $F$ due to the modifications. It is important to reverse the specific position back to its original character after the modifications, since only one position can be modified at a time.

If any moo substring ever reaches a frequency of $F$ before or after modifications, it is added to the set of answers.

While technically the time complexity is $\mathcal{O}(N)$, the factor of 26 in this case is significant because it represents the number of letters we brute-force over.

## Implementation

**Time Complexity:** $\mathcal{O}(CN)$, where $C$ is the number of letters in the alphabet ($26$ in this case).

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	int n, f;
	cin >> n >> f;
	string c;
	cin >> c;

	set<string> answers;
	map<string, int> counts;

	// Function to check whether a given sequence starting at position i is a MOO
	auto is_moo = [&](int i) -> bool {
		if (i < 0 || i + 2 >= n) { return false; }
		return (c[i] != c[i + 1] && c[i + 1] == c[i + 2]) ? true : false;
	};

	// Updates a given string's frequency in the map
	auto update = [&](int i, int change) {
		if (!is_moo(i)) { return; }

		string t = c.substr(i, 3);
		counts[t] += change;
		if (counts[t] >= f) { answers.insert(t); }
	};

	// For a certain modification, updates all the affected strings
	auto update_range = [&](int i, int delta) {
		update(i - 2, delta);
		update(i - 1, delta);
		update(i, delta);
	};

	// Add the original counts of the array
	for (int i = 0; i < n - 2; i++) { update(i, 1); }

	// Test all possible modifications
	for (int i = 0; i < n; i++) {
		char original = c[i];

		for (char let = 'a'; let <= 'z'; let++) {
			update_range(i, -1);
			c[i] = let;
			update_range(i, 1);
		}

		update_range(i, -1);
		c[i] = original;
		update_range(i, 1);
	}

	cout << answers.size() << '\n';
	for (const string &moo : answers) { cout << moo << '\n'; }
}
```

```py
n, f = [int(i) for i in input().split()]
c = list(input())

answers = set()
counts = {}


def is_moo(i: int) -> bool:
	""":returns whether a given sequence starting at position i is a MOO"""
	if i < 0 or i + 2 >= n:
		return False
	return c[i] != c[i + 1] and c[i + 1] == c[i + 2]


def update(i: int, change: int):
	"""Updates a given string's frequency in the map"""
	if not is_moo(i):
		return
	t = "".join(c[i : i + 3])
	counts[t] = counts.get(t, 0) + change
	if counts[t] >= f:
		answers.add(t)


def update_range(i: int, delta: int):
	"""For a certain modification, updates all the affected strings"""
	update(i - 2, delta)
	update(i - 1, delta)
	update(i, delta)


# Add the original counts of the array
for i in range(n - 2):
	update(i, 1)

# Test all possible modifications
for i in range(n):
	original = c[i]

	for let in map(chr, range(ord("a"), ord("z") + 1)):
		update_range(i, -1)
		c[i] = let
		update_range(i, 1)

	update_range(i, -1)
	c[i] = original
	update_range(i, 1)

print(len(answers))
print("\n".join(sorted(answers)))
```
