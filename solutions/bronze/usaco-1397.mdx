## Explanation

First, we can directly make an array that contains the difference between $c_i$ and $t_i$, since we eventually need to subtract ti in every case.

Next, we can sort this array in descending order.

Then, we can check if the value located at $V-1$ in the array is greater than $S$. It means that following elements is possible, since it is sorted in descending order. In other words, going $V$ farms is possible when the value at $V-1$ is greater than $S$.

Lastly, we can print the answer according to the comparison.

## Implementation

**Time Complexity:** $\mathcal{O}(N \log N + Q)$

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    int n,q; cin >> n >> q;
    int c[n]; for (int i = 0; i < n; i++) { cin >> c[i]; }
    for (int i = 0; i < n; i++) { int tmp; cin >> tmp; c[i] -= tmp; }

    sort(c,c+n,greater<int>());

    for (int r = 0; r < q; r++) {
        int v,s; cin >> v >> s;
        if (c[v-1] > s) { cout << "YES\n"; }
        else { cout << "NO\n"; }
    }

    return 0;
}

```

<JavaSection>

```java
import java.util.*;

public class MaximizingProduct {
    public static void main(String[] args) throws Exception {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int q = sc.nextInt();

        Integer[] c = new Integer[n];
        for (int i = 0; i < n; i++) { c[i] = sc.nextInt(); }
        for (int i = 0; i < n; i++) { c[i] -= sc.nextInt(); }
        Arrays.sort(c,Collections.reverseOrder());

        for (int r = 0; r < q; r++) {
            int v = sc.nextInt();
            int s = sc.nextInt();

            if (c[v-1] > s) { System.out.println("YES"); }
            else { System.out.println("NO"); }
        }
        sc.close();
    }
}
```
