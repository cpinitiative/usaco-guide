[USACO Guide Groups](/groups) is a learning management system that allows you to
post announcements and assign problems to group members. It also includes a live
leaderboard to encourage competition! Access is now available to the public; if
you want to create a group, fill out
[this form](https://docs.google.com/forms/d/e/1FAIpQLSemoYwALeum82x_emoZcliKgTiAjHJdJJVKJTkHIWgxp5NPag/viewform).
